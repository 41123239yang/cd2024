# CoppeliaSim 的操作 (足球場)
足球場景, 要導到 CoppeliaSim ,要先找到自己的 ipv4 的網路, 關掉防火牆, 也要把內部程式的網路改成對應的 ipv , 才能執行虛擬網頁操作. 但我遇到的問題是, 網路有改, 防火牆也有關, 就是跑不出來, 是失敗的. (前提是老師有開啟共用 ip 的情況下操作)
